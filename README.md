🗣️ Echo Chat-Bot – Multi‑Language Edition

A simple **echo chat‑bot** that repeats what you type, with a few extra commands like greeting, time, help, and conversation history.  
Built in **7 programming languages** – perfect for learning basic I/O, string processing, and interactive loops.

## ✨ Features
- **Echo mode** – repeats your message exactly (or with a twist).
- **Commands** – `/help`, `/time`, `/greet`, `/history`, `/bye`.
- **History** – remembers the last 20 messages.
- **Personality** – responds with a random greeting or playful variation (optional).
- **Case flip** – `/flip <message>` echoes in inverted case.
- **Colored output** – optional ANSI colors for fun.
- **Cross‑platform** – works on any terminal.

## 🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

| Language | Command |
|----------|---------|
| Python   | `python echo_bot.py` |
| Go       | `go run echo_bot.go` |
| JavaScript | `node echo_bot.js` |
| C#       | `dotnet run` (or `csc EchoBot.cs && EchoBot.exe`) |
| Java     | `javac EchoBot.java && java EchoBot` |
| Ruby     | `ruby echo_bot.rb` |
| Swift    | `swift echo_bot.swift` |

## 🔧 Commands
| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/greet` | Get a random greeting |
| `/time` | Display current date and time |
| `/history` | Show recent messages |
| `/flip <msg>` | Echo with swapped case |
| `/bye` | Quit the bot |
